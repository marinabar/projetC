# Installation et démarrage du jeu
Pour mettre en place le jeu, lancer


```bash
gcc -o bestgame main.c src/*.c -Isrc -Wall
```
Félicitions ! Vous pouvez dès à présent jouer avec

```bash
./bestgame
```
